## ✨ Fonctionnalités

- Design responsive (mobile, tablette, desktop)
- Navigation fixe avec effet de transparence au scroll
- Animations CSS au scroll via IntersectionObserver
- Barres de compétences animées
- Formulaire de contact avec validation JavaScript
- Menu hamburger pour mobile
- Smooth scroll sur toutes les ancres

## 🛠️ Stack technique

| Technologie | Usage |
|-------------|-------|
| HTML5 sémantique | Structure de la page |
| CSS3 (Flexbox, Grid, Variables) | Mise en page et animations |
| JavaScript ES6 (vanilla) | Interactivité, validation formulaire |
| Google Fonts | DM Serif Display + DM Sans |

## 🌍 Déployer sur Vercel

1. Va sur [vercel.com](https://vercel.com) et connecte ton GitHub
2. Importe ce repo
3. Clique sur **Deploy** — c'est en ligne !

---

## 📬 Configurer le formulaire (Formspree)

1. Crée un compte sur [formspree.io](https://formspree.io)
2. Crée un formulaire et copie ton ID
3. Dans `index.html`, remplace `TON_ID` dans l'action du formulaire :
   ```html
   action="https://formspree.io/f/TON_ID"
   ```
